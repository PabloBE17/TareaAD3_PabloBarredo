Inicio de Sesión

Este módulo permite a los usuarios autenticarse en el sistema con sus credenciales.

Funcionamiento

Cuando el usuario ingresa su nombre de usuario y contraseña, el sistema valida las credenciales y decide qué tipo de usuario ha iniciado sesión:

  • Administrador: Si las credenciales coinciden con las almacenadas en el archivo Admin.properties, se redirige al menú de administrador.
  • Peregrino: Si el usuario está registrado como peregrino, se obtiene su ID y se redirige al menú de peregrino.
  • Administrador de Parada: Si el usuario pertenece a una parada, se obtiene su ID y se muestra el menú correspondiente.
  • Error: Si las credenciales son incorrectas, se muestra un mensaje de error.

Otras Funcionalidades

  • Permite alternar la visibilidad de la contraseña.
  • Redirige a la vista de registro de peregrinos si el usuario lo desea.

Mensajes de Alerta

El sistema muestra diferentes alertas según la acción realizada:

  • Campos Vacíos: Si el usuario no completa los campos.
  • Bienvenida: Si las credenciales son correctas.
  • Error de Autenticación: Si las credenciales son incorrectas.

Seguridad

Las credenciales de administrador se almacenan en un archivo Admin.properties, mientras que el resto de usuarios se verifican con el servicio UsuarioServicio.
